## 추가 패키지
- styled-components
    - npm install styled-components
    - version: 5.3.6

- react-router-dom
    - npm install react-router-dom --save
    - version: 6.4.2

- firebase
    - npm install firebase
    - version: 9.15.0

## 실행 방법
